# 🛋️ SofaStream

**SofaStream** — это веб-платформа для совместного просмотра видеоконтента в реальном времени, созданная с фокусом на пары, находящиеся на расстоянии. 

В отличие от трансляции экрана в мессенджерах, SofaStream обеспечивает **жесткую синхронизацию состояния плеера** у всех участников комнаты, компенсируя сетевые задержки и автоматически реагируя на буферизацию (падение скорости интернета) у отдельных пользователей.

*(🚧 Проект находится в стадии активной разработки 🚧)*

## Архитектура и паттерны

Проект спроектирован с упором на качество кода, тестируемость и масштабируемость. В основе лежит **Clean Architecture** и принципы **Domain-Driven Design (DDD)**.

### Ключевые особенности реализации:
- **Rich Domain Model:** Вся логика синхронизации, управления участниками и назначения хоста инкапсулирована в `Aggregate Root` (`Room.cs`).
- **Hybrid ORM (CQRS на уровне БД):** 
  - **Entity Framework Core:** используется для команд (записи), обеспечивая надежное сохранение сложных агрегатов и работу с миграциями.
  - **Dapper:** используется для запросов (чтения), обеспечивая максимальную производительность при получении состояния комнаты через прямой SQL.
- **Result Pattern:** Вместо исключений для управления ожидаемыми бизнес-ошибками используется функциональный подход с объектом `Result`, что делает API предсказуемым и безопасным.
- **Domain Events:** Изменения в домене порождают события (например, `RoomPlaybackStateChangedEvent`), которые транслируются в реальном времени через SignalR.
- **Global Error Handling:** Реализован централизованный перехватчик исключений (`IExceptionHandler`), предотвращающий утечку технических данных и гарантирующий JSON-ответ при любых сбоях.
- **Smart Disconnect:** Автоматическое отслеживание разрывов соединений в SignalR Hub с переназначением прав хоста, если лидер покинул сессию.

## 🛠 Стек технологий
- **Backend:** C# 14, .NET 10, ASP.NET Core (Web API)
- **Architecture:** Clean Architecture, DDD, CQRS
- **Database:** PostgreSQL (Npgsql).
- **ORM:** Entity Framework Core & Dapper.
- **Real-time:** SignalR (WebSockets)
- **Environment:** Docker Compose для быстрой развертки БД.
- **Frontend:** *В планах (React / Vue.js)*

## 🚀 Быстрый запуск (Backend)

Проект еще находится в стадии разработки, но если вы хотите, то можете опробовать API уже сейчас.

Шаги для запуска:
1. Убедитесь, что установлен **Docker Desktop**.
2. В корне проекта выполните:
    ```bash
    docker-compose up -d
    ```
3. Примените миграции:
    ```bash
    dotnet ef database update --project SofaStream.Infrastructure --startup-project SofaStream.Api
    ```
4. Запустите API и откройте Swagger по адресу http://localhost:5063.
